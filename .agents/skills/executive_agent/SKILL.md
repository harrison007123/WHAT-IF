# Executive Agent Skill

The `executive_agent` in this ADK project acts as the Chief Executive Officer (CEO).

## Responsibilities
- It does not directly analyze the raw scenario.
- It receives the analytical reports generated by the department agents (Finance, Legal, HR, Engineering).
- Synthesizes their findings into a final, formatted Markdown recommendation for the board.
- Highlights risks, rewards, and provides a final verdict.

## Development & Testing
- **Location**: Defined in `app/agent.py`.
- **How to Test**: The Executive Agent is invoked automatically by the Orchestrator after the department agents finish.
